\# NEXUS



A local AI command center powered by Ollama.



\## Features



\* Local AI models with Ollama

\* Model selection

\* Rich terminal UI

\* Memory system

\* File tools

\* Terminal tools

\* Developer-focused workflow



\## Requirements



\* Python 3.10+

\* Ollama



\## Installation



```bash

pip install -r requirements.txt

```



\## Run



```bash

python main.py

```



\## Supported Models



\* qwen2.5

\* gemma4

\* gemma3

\* phi
